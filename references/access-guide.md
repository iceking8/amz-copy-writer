# CLR 访问指南

## 什么是 Category Listing Report

Amazon Seller Central 的详细报告，提供：
- **Product Classification**：产品分类是否正确（分类错误会严重影响曝光和销量）
- **Product Titles and Descriptions**：listing 在搜索结果中的呈现方式
- **Backend Keywords**：与 listing 关联的搜索词及其表现
- **Pricing and Competition**：与竞品的价格对比
- **Performance Metrics**：特定类目下的销量、点击、转化

## 为什么重要

Amazon 搜索算法高度依赖结构化和相关数据。使用此报告可以：
- **发现优化机会**：优化关键词、标题、描述以提升可发现性
- **监控趋势**：把握季节性变化和类目动态
- **超越竞品**：基于竞品洞察调整策略

## 访问限制原因

- **敏感数据**：包含产品表现、关键词、竞品信息等专有数据
- **专属用途**：仅限积极优化 listing 的活跃卖家
- **高级洞察**：需要专业知识才能充分利用

## 资格要求

1. **专业卖家账户**：个人卖家账户不符合资格
2. **相关类目有活跃 listing**：必须有活跃 listing，incomplete listing 不算
3. **Amazon Brand Registry**：品牌备案卖家优先
4. **账户权限**：可能需要管理员账户权限

## 如何申请访问

Seller Central → Reports → Inventory reports → Category listing reports → Request report

注：个人卖家需联系支持获取一次性访问权限。
