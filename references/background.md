# Category Listings Report 背景知识

## 是什么

Category Listings Report (CLR) 是亚马逊卖家中心的报告，可下载现有 contribution SKUs 的 Excel 电子表格。报告可编辑后上传更新 listings。

## 谁能用

- **专业卖家账户**：默认有权限
- **个人卖家**：无权限，需联系支持获取一次性访问（获取后无时间限制）

## 如何获取

Seller Central → Reports → Inventory reports → Category listing reports → Request report

## 报告包含

- 最新提交数据（无论用什么工具创建/编辑）
- 所有产品和报价数据（价格、数量等）
- 只包含提交的数据，不包含 ASIN 其他关联数据
- 默认包含 active / inactive / incomplete 状态的 listings
- 不包含已删除的 contribution SKUs

## 格式

- Excel (.xlsx) 或 zip 文件（多个电子表格）
- 大型报告：可能分成多个电子表格，合并成单个 zip
- Beta 版新电子表格有更多功能

## 注意事项

- 生成报告最多需要三天
- 大型 SKU 列表：数据可能分成多个电子表格
- Beta 和当前电子表格会分开，确保每个文件不超过 30 个产品类型 and 10,000 行
- 报告生成期间无需停留在页面，稍后返回下载即可
