# Amazon 商品信息规则与限制

> 来源：Subscribe With Amazon 计划商品信息指南

## 创建商品信息的规则与限制

### 质量要求
- 提供给亚马逊的商品信息应至少与卖家销售渠道中显示或使用的最高质量信息保持相同的质量水平

### 禁止行为
- ❌ 使用虚假的商品编码信息
- ❌ 一个详情页面发布多种商品
- ❌ 推广"受限商品"

### 禁止包含的内容

商品信息页面的商品名称、描述、要点或图片中**不得包含**：

| 禁止内容 | 说明 |
|----------|------|
| 色情、淫秽或诽谤内容 | — |
| 电话号码、实际邮寄地址、电子邮件地址 or 网站 URL | 必填商品信息的特定 URL 除外 |
| 备选订购信息 | 如转至其他网站下单的链接 |
| 评论、引用或推荐内容 | — |
| 恳请买家发布正面评价的言辞 | — |
| 图片/照片/视频上的广告、宣传资料或水印 | — |
| 时间敏感信息 | 如宣传活动、研讨会、讲座等活动的日期 |

### 禁止使用的数据格式

商品名称、描述、要点及任何其他属性中**不得使用**：

| 禁止类型 | 示例 |
|----------|------|
| HTML 标签 | `<br>`, `<p>`, `<b>` 等 |
| 特殊字符 | `*`, `/`, `?` 等 |

### 图片标准
- 所有商品图片必须符合亚马逊的图片标准
- 数字化图像应仅包含商品本身
- 不包含任何多余徽标、文字或其他标记

---

## 审查检查项对照

以上规则对应 CLR 审查的以下检查项：

| 检查项 | 对应规则 |
|--------|----------|
| 标题违禁字符 | 不得使用 HTML、特殊字符 */? 等 |
| 任意字段违禁字符 | 同上，全字段扫描 |
| Bullet 违禁内容 | 不得包含电话、邮箱、URL、评论、推荐内容等 |
| Bullet 软性问题 | 时间敏感信息、营销废话 |

## 必填商品信息清单

创建商品详情时需提供：

- [ ] 商品名称
- [ ] 商品分类
- [ ] 订阅长度（时长和/或订阅期间的交付数量）
- [ ] 购买价格
- [ ] SKU 和 UPC/EAN/JAN 编号
- [ ] 兼容设备
- [ ] 支持的登录账号类
- [ ] 供货情况相关信息
- [ ] 商品数字化图像
- [ ] 法律要求的文字、免责声明、警告、通知、标签
- [ ] 您的网站链接
- [ ] 隐私政策链接
- [ ] 其他相关条款和条件的链接

## 自动续订商品额外要求

如果商品为自动续订订阅服务，需显示：

- 未在周期结束前 24 小时关闭则自动续订
- 当前周期结束前 24 小时内扣款
- 续订周期和费用
- 买家可在买家订阅设置中管理订阅和关闭自动续订
