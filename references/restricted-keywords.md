# 亚马逊受限关键词列表

> 来源：400+ Restricted Amazon Keywords: The Most Comprehensive List You'll Need

## 概述

受限关键词是亚马逊禁止或限制在产品列表中使用的术语。使用这些关键词可能导致：
- 列表被屏蔽
- 账户暂停
- 永久禁止进入市场

## 亚马逊为什么要限制关键词

| 原因 | 说明 |
|------|------|
| **遵守法律标准** | 防止未经核实的医疗声明、未经许可使用商标 |
| **保护知识产权** | 禁止未经授权使用竞争对手品牌名称或商标 |
| **提升客户体验** | 防止误导性或虚假声明，确保信息准确 |

## 受限关键词类型

| 类型 | 示例 |
|------|------|
| **商标和品牌名称** | 未经许可使用其他公司的品牌名、徽标、标语 |
| **违禁内容** | 成人内容、攻击性语言、不当或有害术语 |
| **医疗声明** | 未经 FDA 批准的健康声明（"治愈癌症"、"预防心脏病"） |
| **比较和最高级术语** | "最好"、"最便宜"、"第一"（除非有可验证证据） |
| **主观术语** | "惊人"、"不可思议"、"必备" |
| **促销术语** | "Free Shipping"、"Discount"、"Sale" |

## 检查建议

### 高风险关键词（必须避免）

1. **医疗声明类**：任何涉及疾病名称、治疗、治愈的词汇
2. **FDA 声明类**：未经 FDA 批准不得使用
3. **Amazon 滥用类**：不得虚假声称与亚马逊有关联
4. **促销类**：Free Shipping、Discount、Sale 等

### 中风险关键词（需上下文判断）

1. **最高级类**：Best、Top、Number One 等
2. **认证类**：Certified、Guaranteed、Proven 等
3. **环保声明**：Eco-friendly、Biodegradable 等

### 检查方法

1. **自动化检查**：在 CLR 审查中扫描这些关键词
2. **上下文判断**：检查关键词使用是否具有误导性
3. **证据要求**：如有证据支持，部分关键词可使用

---

## 违规后果

| 后果 | 说明 |
|------|------|
| **列表屏蔽** | 客户无法找到产品，销售额下降 |
| **账户暂停** | 临时或永久禁止销售 |
| **品牌信任受损** | 负面评论，品牌形象受损 |

---

## 最佳实践

1. **使用清晰、准确的描述**：避免夸张或无法验证的声明
2. **避免未经证实的声明**：除非有确凿证据
3. **定期审核列表**：检查是否无意中使用受限关键词
4. **培训团队**：确保所有相关人员了解政策
